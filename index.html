Wamp Setup

 Posted on January 21, 2026

Heute habe ich auf GitHub meine WAMP Setup Beschreibung veröffentlicht.

WAMP ist eine XAMPP ähnliche Umgebung für Windows.

Auf einem Windows System werden dabei

  • Apache Web Server
  • MariaDB
  • PHP

so konfiguriert, dass alle Daten in einem Verzeichnis abgelegt sind. Keine der Komponenten wird installiert oder als Service betrieben.
